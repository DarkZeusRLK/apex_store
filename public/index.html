Workflow

SCRIPTS FIVEM

Entenda o processo de criação, do ticket à entrega final.

Passo 01

Inicie seu Projeto no Discord

Tudo começa em nosso Discord Oficial. Para solicitar um sistema exclusivo, você deve abrir um ticket de orçamento na categoria correspondente.

1. Entre no Discord da Apex Store
2. Vá até o canal #tickets
3. Selecione a opção "Solicitar Script Personalizado"
Passo 02

Briefing e Estilização

Nesta etapa, você descreve a lógica do seu script. É fundamental detalhar como o sistema deve funcionar e como deve ser a aparência da interface (NUI).

  • Funcionalidades: Regras de negócio e comandos.
  • Design: Cores, formas e layout preferido.
Passo 03

Desenvolvimento e Testes

Após o envio das informações, os desenvolvedores DARK e TONIN iniciam a codificação do projeto.

Garantia de Performance: O script passará por testes de estresse para garantir que rode com o menor consumo de CPU possível (0.01ms).

Passo 04

Pagamento e Recebimento

Com o script pronto, faremos uma demonstração (vídeo ou live). Assim que você aprovar o resultado final:

1. Você efetua o pagamento via o ticket aberto
2. O código-fonte ou licença é enviado instantaneamente
3. Nossa equipe auxilia na instalação inicial, se necessário

Dúvidas Frequentes

Como recebo meu produto?
Após o pagamento, o download é liberado automaticamente via bot no nosso Discord.
Os scripts são otimizados?
Sim! Todos os nossos sistemas rodam a 0.01ms ou menos, garantindo o melhor FPS para o seu servidor.
Onde posso tirar dúvidas?
Diretamente em nosso Discord! Basta abrir um ticket e estaremos te atendendo.
A APEX corrige bugs e faz manutenção?
Sim para ambos! Após o pagamento, o cliente deverá testar o produto e se encontrar bugs ou falhas deve reportar em um ticket via discord imediatamente para a correção. A Manutenção é cobrada de acordo com a complexidade do sistema, o valor é passado juntamente do valor final do script, a Manutenção é opcional.
