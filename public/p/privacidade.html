Política de Privacidade

Como lidamos com seus dados.

1. Coleta de Dados

Para a realização de orçamentos e entrega de produtos, coletamos apenas informações essenciais via Discord, como seu ID de usuário e detalhes do projeto solicitado.

2. Uso das Informações

Os dados fornecidos no formulário de orçamento são enviados via Webhook para nosso servidor interno apenas para fins de contato e faturamento. Não compartilhamos esses dados com empresas externas.

3. Segurança

Todos os scripts são testados para garantir que não existam vulnerabilidades (backdoors) que possam comprometer a segurança do seu servidor.
